Detailed Description

Handle distortion related stuff.

Distortion correction can be used to solve problems resulting from an uneven build plate. It allows measuring a nxn grid with a z-probe and add these correction to all moves. Normally you start at the bottom with 100% correction and at 0.5mm you start reducing correction until it vanishes completely at 1-3 mm.

The stored values are steps required to reach the bumped level assuming you are at zMin. So if you have a 1mm indentation it contains -steps per mm.
