143 /*
144 If the firmware is in laser mode, it can control a laser output to cut or engrave materials.
145 Please use this feature only if you know about safety and required protection. Lasers are
146 dangerous and can hurt or make you blind!!!
147 
148 The default laser driver only supports laser on and off. Here you control the eíntensity with
149 your feedrate. For exchangeable diode lasers this is normally enough. If you need more control
150 you can set the intensity in a range 0-255 with a custom extension to the driver. See driver.h
151 and comments on how to extend the functions non invasive with our event system.
152 
153 If you have a laser - powder system you will like your E override. If moves contain a
154 increasing extruder position it will laser that move. With this trick you can
155 use existing fdm slicers to laser the output. Laser width is extrusion width.
156 
157 Other tools may use M3 and M5 to enable/disable laser. Here G1/G2/G3 moves have laser enabled
158 and G0 moves have it disables.
159 
160 In any case, laser only enables while moving. At the end of a move it gets
161 automatically disabled.
162 */
