238 If the firmware is in laser mode, it can control a laser output to cut or engrave materials.
239 Please use this feature only if you know about safety and required protection. Lasers are
240 dangerous and can hurt or make you blind!!!
241 
242 The default laser driver only supports laser on and off. Here you control the intensity with
243 your feedrate. For exchangeable diode lasers this is normally enough. If you need more control
244 you can set the intensity in a range 0-255 with a custom extension to the driver. See driver.h
245 and comments on how to extend the functions non invasive with our event system.
246 
247 If you have a laser - powder system you will like your E override. If moves contain a
248 increasing extruder position it will laser that move. With this trick you can
249 use existing fdm slicers to laser the output. Laser width is extrusion width.
250 
251 Other tools may use M3 and M5 to enable/disable laser. Here G1/G2/G3 moves have laser enabled
252 and G0 moves have it disables.
253 
254 In any case, laser only enables while moving. At the end of a move it gets
255 automatically disabled.
256 */
