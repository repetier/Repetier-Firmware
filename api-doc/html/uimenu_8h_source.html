30 /*
31  The menu configuration uses dynamic strings. These dynamic strings can contain
32  a placeholder for special values. During print these placeholder are exchanged
33  by their current value. Everything else is printed exactly as written.
34 
35  A placeholder always has 3 chars. It starts with a % followed by 2 characters
36  defining the value. You can use any placeholder in any position, also it doesn't
37  always make sense.
38 
39  Special Characters
40  constant description
41  cUP Folder up arrow
42  cDEG Degree mark
43  cSEL Selected
44  cUNSEL Unselected
45  cTEMP Thermometer symbol
46  cFOLD Folder symbol
47 
48  List of placeholder:
49  %%% : The % char
50  %% : The % char (also)
51 
52  %?<c> : Conditional. Print c if current char is not c. Allows avoiding duplicate character, like space
53 
54  acceleration
55  %ax : X acceleration during print moves
56  %ay : Y acceleration during print moves
57  %az : Z acceleration during print moves
58  %aX : X acceleration during travel moves
59  %aY : Y acceleration during travel moves
60  %aZ : Z acceleration during travel moves
61  %aj : Max. jerk
62  %aJ : Max. Z-jerk
63 
64  debug
65  %do : Debug echo state.
66  %di : Debug info state.
67  %de : Debug error state.
68  %dd : Debug dry run state.
69  %dp : Debug end stop state.
70  endstop stats with only one char
71  %dx : Debug min x endstop
72  %dX : Debug max x endstop
73  %dy : Debug min y endstop
74  %dY : Debug max y endstop
75  %dz : Debug min z endstop
76  %dZ : Debug max z endstop
77 
78  extruder
79  %ec : Current extruder temperature
80  %ed : Number of copies for ditto mode
81  %eIc : Current extruder temperature integer (shorter)
82  %eb : Current heated bed temperature
83  %e0..9 : Temp. of extruder 0..9

86  %Ec : Target temperature of current extruder

109  %oe : Error message
